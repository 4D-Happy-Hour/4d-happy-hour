# 4D Happy Hour static site

Static website files for https://4dhappyhour.com.

## Update the Zoom link

In `index.html`, replace:

```html
PASTE_ZOOM_LINK_HERE
```

with the actual Zoom meeting URL.

## GitHub Pages

This can be served from the repository root, or from `/docs` if the repo is configured that way.

Files:

- `index.html`
- `styles.css`
- `assets/4dhh-logo.svg`
- `assets/4dhh-logo-bw.svg`
- `assets/4dhh-ad-square.png`
